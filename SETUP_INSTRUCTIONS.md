# Portfolio Setup Instructions

## Files to Add

### 1. Profile Picture
- **File name:** `profile.jpg`
- **Location:** Place in the root directory (same folder as `index.html`)
- **Requirements:**
  - Professional headshot
  - Recommended size: 350x350px or larger (square format works best)
  - Formats: JPG, PNG, or WebP
  - The image will be automatically styled and displayed in the hero section

### 2. CV/Resume PDF
- **File name:** `cv.pdf`
- **Location:** Place in the root directory (same folder as `index.html`)
- **Requirements:**
  - PDF format
  - Professional resume document
  - The download button will automatically link to this file

## Color Scheme
The portfolio uses a professional **Blue + White + Gray** color scheme:

## Customization

### To change profile picture:
1. Replace `profile.jpg` with your image
2. Keep the same filename or update the `src` attribute in `index.html` line 40

### To update CV:
1. Replace `cv.pdf` with your resume
2. Keep the same filename or update the `href` attributes in `index.html`

### To modify colors:
Edit the CSS variables in `styles.css` at the top of the file (lines 2-20)
